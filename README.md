# Employee CRUD App

This is a simple web application built using **Node.js**, **Express**, **MongoDB**, and **EJS** that allows you to perform CRUD (Create, Read, Update, Delete) operations on employee data.

## Features

- Add a new employee
- View all employees
- Edit existing employee details
- Delete an employee

## Technologies Used

- Node.js
- Express.js
- MongoDB (Mongoose)
- EJS
- Bootstrap 5
